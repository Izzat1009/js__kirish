Data types




  • * 1. Primitive
  • * 1.1 number - 1, -5, 0 1.10-gacha bo'lgan sonlarda foydalaniladi.
  • * 1.2 string - "" Kerakli malumotlar predmet, yana hokazo kerak bo'lganda foydalaniladi va bu code qo'shtirnoq yani "" shuni ichiga yoziladi
  • * 1.3 boolean - true, false Misol uchun: Sayohat qilganmisiz "Xa" yoki "yo'q" shunday narsalarda foylaniladigan code, bu codeda hechqanday "" , qilinmaydi faqat true yoki false

  • * 1.5 null - object Null object, bu code Misol quti bor ichida hech nima yo'q u quti lekin sizniki shunday manoni anglatadi, bu codeda bir box ni band qilib qo'yishingiz mumkin bo'ladi
  • * 1.6 symbol Symbol, Tenglikni anglatuvchi code.
  • * 1.7 bigint Bigint bu code Tasodifiy har qanday raqamni ohiriga "n" deb yozib qo'yasiz...
